<task type="auto">
  <name>Task 1: Create vercel.json with SPA rewrite rule</name>
  <files>vercel.json</files>
  <action>
    Create `vercel.json` at the repository root with an Angular SPA rewrite rule. Every URL path must resolve to `index.html` so Angular's router handles client-side routing.

    Content:
    ```json
    {
      "rewrites": [{ "source": "/(.*)", "destination": "/index.html" }]
    }
    ```

    This is required because Angular is a SPA — without it, navigating directly to `/board` on Vercel returns a 404 instead of loading the app.

    Do NOT add any build command or outputDirectory here — those are configured in Vercel's project settings dashboard (not in vercel.json) to avoid conflicts with the free tier configuration.
  </action>
  <verify>
    <automated>test -f /Users/peterichardson/Repos/Carno-Frontend-Engineer-Exercise/vercel.json && grep -q "rewrites" /Users/peterichardson/Repos/Carno-Frontend-Engineer-Exercise/vercel.json && echo "vercel.json valid" || echo "FAIL"</automated>
  </verify>
  <done>vercel.json exists at repo root containing a `rewrites` array with the index.html destination pattern</done>
</task>

    Steps to perform:

    2. Click "Import Git Repository" and select the Carno-Frontend-Engineer-Exercise repo
